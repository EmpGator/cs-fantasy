This is a README file.


This project is meant to be substitute for google sheets based approach on counter strike pick em fantasy league.
Envisioned tech stack includes:
- Python
- Django
- PostgreSQL (probably dockerized)
- htmx
- some frontend library for components
- maybe some reacitive prop library


Goal is to make MVP with similar functionality to google sheets asap and then improve it with more features and better user experience.
